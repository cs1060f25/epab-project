# CS12-1 Prototype 4: Jupyter Notebook Interactive

**Incident Timeline & Evidence Assembly with Natural Language Assistant**

## Overview
An investigation notebook that combines data analysis, visualization, and documentation in a single exploratory interface. Designed for data-savvy SOC analysts who want to apply statistical methods and create comprehensive investigation reports.

## Design Dimensions
- **Tech Stack**: Python + Jupyter + Pandas + Plotly + Matplotlib
- **Design Language**: Notebook-based, exploratory
- **User Type**: Data-savvy analyst who codes
- **Interaction**: Code cells + markdown documentation
- **Architecture**: Notebook with embedded visualizations

## Features
- Pre-populated cells with common investigation queries
- Interactive Plotly timeline visualization
- Natural language search via f-strings and pandas queries
- Statistical anomaly detection and scoring
- Correlation analysis between events
- Automated report generation
- JSON export functionality
- Executable and modifiable code cells
- Rich markdown documentation explaining each step

## What Worked

### Strengths
- **Exploratory Analysis**: Perfect for hypothesis-driven investigation
- **Documentation**: Combines analysis with narrative explanation
- **Reproducibility**: Investigation steps are preserved and repeatable
- **Flexibility**: Code can be modified for different incident types
- **Rich Visualizations**: Interactive plots reveal patterns
- **Statistical Methods**: Quantitative approach to anomaly detection
- **Report Generation**: Automated professional reporting
- **Educational**: Shows methodology and reasoning

### Professional Benefits
- **Audit Trail**: Every analysis step is documented
- **Collaboration**: Notebooks can be shared and peer-reviewed
- **Methodology**: Transparent analytical approach
- **Extensibility**: Easy to add new analysis techniques
- **Integration**: Can import data from multiple sources
- **Export**: Results easily shared with stakeholders

This prototype demonstrates how data science techniques can enhance security investigation workflows while maintaining full transparency and reproducibility.
